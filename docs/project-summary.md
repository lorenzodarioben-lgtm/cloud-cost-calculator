# Project Summary

Cloud Cost Calculator is a lightweight AWS estimate mockup for modelling simple EC2 and EBS monthly costs.

The project demonstrates:

- cloud cost awareness
- EC2 hourly compute estimation
- EBS GB-month storage estimation
- budget guardrail messaging
- static frontend development
- JavaScript calculation logic
- automated validation with Node tests

## Purpose

The goal is not to replace the official AWS Pricing Calculator. Instead, this project is a focused learning tool and portfolio project that shows how cloud cost components can be modelled clearly in a small frontend application.

## Portfolio value

This project is useful for demonstrating practical understanding of:

- AWS cost basics
- FinOps thinking
- simple UI state management
- defensive input handling
- documentation
- GitHub Actions validation
