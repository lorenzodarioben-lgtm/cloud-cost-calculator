\# Pricing Assumptions



This project is a simplified AWS cost estimator mockup. It is designed for learning, portfolio demonstration, and quick estimation practice.



\## Default region



The default pricing assumptions are based on:



\- Region: US East (N. Virginia), `us-east-1`

\- Operating system: Linux/Unix

\- Currency: USD
