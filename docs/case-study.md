# Portfolio Case Study

## Problem

Cloud costs can be hard to reason about when learners are first introduced to AWS. Even simple services like EC2 and EBS have pricing units that are not always obvious at first glance.

## Solution

This project presents a small interactive calculator that breaks the estimate into clear parts:

- compute cost
- storage cost
- monthly total
- budget status

## Technical approach

The app uses plain HTML, CSS, and JavaScript. The calculator logic is separated into pure functions so it can be tested independently from the UI.

## What I practised

- structuring a small frontend project
- writing reusable calculation functions
- handling invalid user input
- creating a polished responsive interface
- documenting assumptions clearly
- using GitHub Actions for validation

## Future direction

The project could be expanded with more AWS services, comparison mode, region switching, and exportable estimates.
