# Example Scenarios

The `scenarios.json` file contains example calculator inputs that can be used for testing, demos, and future UI improvements.

## Included scenarios

- Tiny student project
- Always-on portfolio backend
- Growing project

## Why scenarios matter

Scenarios make it easier to explain what the calculator does without manually inventing inputs each time.

They can also be used later for:

- preset cards
- automated regression tests
- README examples
- demo screenshots
- comparison mode
